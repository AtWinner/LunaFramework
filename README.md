# LunaFramework

一款Android快速开发框架

# 导入

```
allprojects {
    repositories {
        jcenter()
        maven { url "https://github.com/AtWinner/LunaFramework/raw/master/" }//
    }
}
```

```
dependencies {
    compile 'com.luna:base_framework:1.0.0'
    compile 'com.luna:view_framework:1.0.0'
}
```
